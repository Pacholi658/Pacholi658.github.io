About Our Data

Data Overview

The dataset we utilized is titled NBA Shots 2024, sourced from Kaggle. This dataset, created by user Kaggle contributor mexwell, contains NBA players' regular-season shooting data spanning from the 2003-04 to the 2023-24 seasons. Each dataset provides detailed information about every shot taken in a single NBA season, including:

  • Team, player, and position
  • Game details
  • Shot-specific data such as distance, zone, and result

Key details about the dataset include:

  • Original dataset size: 218,701 rows and 26 columns
  • Filtered Celtics data: 7,396 rows and 26 columns
  • Focus: Data from the 2023-24 NBA season

You can view the dataset directly on Kaggle here: NBA Shots 2024 .
